Програма для обліку продажу квартир. Квартири описуються метражем, ціною, місцем розташування. Також до квартири долучена схема (малюнок). Також можна виставити відсотки премії при продажі квартири та редагувати ріелторів. Розробити просту програму з використанням EF (code first, атрибути), WebAPI. Клієнт WPF може бути реалізований без використання MVVM і мати найпростіший інтерфейс.
